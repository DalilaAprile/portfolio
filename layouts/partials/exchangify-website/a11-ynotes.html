Notes on accessibility

In creating this responsive website aimed at assisting travelers with currency exchange rates, locating nearby ATMs, and tracking currency fluctuations, we prioritized accessibility to ensure a seamless experience for users of all ages and abilities.

Clear Labels with Icons: Essential features like Graphs, ATMs, and Alerts are presented with both icons and text labels. This combination aids users with cognitive disabilities and enhances screen reader compatibility by clearly conveying the function of each button.

Responsive Design: The layout adapts fluidly to various screen sizes, including mobile devices and tablets Text and buttons resize appropriately, maintaining readability and usability across all devicesâ€”a crucial aspect for our diverse user base.

High Contrast Ratio: Interactive elements have a contrast ratio of 3:36, complying with WCAG (Web Content Accessibility Guidelines) standards for larger text and graphics. This ensures that buttons and text are easily visible to users with mild visual impairments, enhancing overall accessibility.

Screen Reader Support with Annotations: The website is optimized for screen readers, incorporating annotations that define the linear focus order. This assists assistive technologies in navigating through forms efficiently, improving the user experience for those relying on screen readers.

By integrating these accessibility features, the website provides an inclusive platform that empowers travelers to make informed financial decisions effortlessly, regardless of their physical abilities or the devices they use.
